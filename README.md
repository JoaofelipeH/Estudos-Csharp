# 📚 Estudos de C# & Fundamentos de Programação

Este repositório é dedicado à prática constante de lógica e fundamentos de desenvolvimento utilizando o ecossistema **.NET**. 

O foco aqui não são projetos complexos, mas a **base sólida**: entender como os dados fluem, como a memória é tratada e como estruturar algoritmos eficientes, fugindo das abstrações do low-code.

---

## 🛠️ Ambiente de Desenvolvimento
* **Linguagem:** C# (Strongly Typed)
* **Runtime:** .NET 8 / .NET 9
* **OS:** Windows (Desenvolvimento via Visual studio)
* **Plataforma de Estudo:** Microsoft Learn & Desafios de Lógica

---

## 📂 Estrutura do Repositório

* `/Logica-Basica`: Variáveis, tipos de dados, condicionais (`if/else`, `switch`) e operadores.
* `/Estruturas-Repeticao`: Práticas com `for`, `while` e `foreach`.
* `/Manipulacao-Dados`: Arrays, Listas e manipulação de Strings.
* `/Metodos-Funcoes`: Organização de código e reaproveitamento de lógica.
* `/Poo-Inicial`: Primeiros passos com Classes, Objetos e Propriedades.
